animation-direction: ;

How the Animation is Played
  • Normal - (Default) It plays from 0% to 100%
  • Reverse - It plays the opposite direction, from 100% to 0%
  • Alternate - Animation first run forward, then backwards & backwards
  • Alternate reverse - Animation first run backwards, then forward & backwards

This uses Name, Duration, Delay, Iteration-count and Direction Values
